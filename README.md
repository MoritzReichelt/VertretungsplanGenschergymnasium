# Vertretungsplan Genschergymnasium

* Schnelleres Abrufen des Vertretungsplans im Vergleich zur Webseite
* Benachrichtigung bei neuem Vertretungsplan
* Anpassung der Darstellung z.B. an Tablets
* Offline-Abruf
